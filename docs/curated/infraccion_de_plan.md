# Infracción de plan y cursos permitidos

- Si el sistema marca infracción de plan o no permite matricular pese a cumplir requisitos, no inventar la causa.
- Según el histórico, algunos casos pueden requerir pedir excepción para que el curso aparezca en la lista de cursos permitidos.
- Para solicitudes de excepción o trámites de matrícula personal, orientar a Campus Virtual PUCP: https://campusvirtual.pucp.edu.pe/.
- Si el problema persiste, derivar a Dirección de Carrera o a matricula-ocr@pucp.edu.pe según corresponda.
- No usar "coordinador de carrera" como derivación genérica si ese cargo no aparece explícitamente para el caso.
