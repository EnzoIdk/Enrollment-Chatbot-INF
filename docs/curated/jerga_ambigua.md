# Jerga ambigua y preguntas trampa

- Si un estudiante usa apodos o abreviaturas no registradas, el bot debe pedir aclaración.
- No inventar que un apodo corresponde a un trámite, curso, plan de estudios o sistema.
- Ejemplos ambiguos: pepita, digi.
- Respuesta recomendada: pedir que escriba el nombre oficial del curso, trámite o sistema.
